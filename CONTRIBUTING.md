## How to Contribute

### 1. Adding a new Trick
1. Fork this repository.
2. Create a new `.md` file in the appropriate folder (e.g., `sound-design/my-trick.md`).
3. **Important:** Add the OS Version Badge at the very top of your file.
4. Use clear headings and bullet points.
5. Submit a Pull Request.

### 2. Style Guide
* **Filenames:** Use kebab-case (e.g., `granular-synthesis-hack.md`).
* **Shortcuts:** Format hardware buttons in bold or code blocks.
  * *Good:* Press **FUNC** + **YES**.
  * *Good:* Press `FUNC` + `YES`.
* **Images:** Keep file sizes small. Store them in the `/assets` folder.

### 3. Reporting Issues
If you find an error (e.g., a trick that no longer works in the new OS), please open an Issue or submit a fix directly.
